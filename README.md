## Arcaea改包简介
改包是指修改安装包的贴图文件，从而实现改变游戏界面的效果，包括但不限于游玩过程的轨道颜色、note颜色、背景图，游玩界面的背景、cg、各种bar的样式、立绘等。目前主流的改包目的大多是获得暗侧背景轨道和note以方便读谱，修改立绘为自己喜欢的样式或者贴图等。
由于改包信息和技术散落在各个群聊和视频中，获取信息和资源较为麻烦。特意编写此教程汇总信息、资源以及教程。
## 声明
目前作者仅制作过`iOS端`的`ipa`改包文件，因此`暂时只维护iOS端改包的信息`。安卓端的改包只会比这更容易，容易得多。后续如果有合作者加入或者作者参与制作了安卓端改包，可能会加入安卓端的教程。
此仓库后续可能考虑维护一些同意开源的改包资源，欢迎各类改包作者提交PR。
